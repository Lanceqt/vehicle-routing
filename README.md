# Vehicle Routing Management System

## Overview

A comprehensive vehicle routing solution designed for service companies with one or multiple vehicles. This system helps
businesses optimize their daily operations by efficiently managing workers, scheduling appointments, and generating
optimized routes for service vehicles.
Perfect for companies providing services such as cleaning, gardening, maintenance, delivery, and any other
location-based business operations.
Features

- **Worker Management**: Easy overview and management of your workforce
- **Appointment Scheduling**: Intuitive scheduling system for customer appointments
- **Vehicle Assignment**: Assign workers to specific vehicles for any given day
- **Route Optimization**: Generate optimized vehicle routes using Google Maps Route Optimization API
- **Multi-Vehicle Support**: Manage fleets of any size with ease
- **Database Storage**: Persistent data storage using SQLite
- **RESTful API**: Clean API interface for integration with other systems
  Technology Stack
- **Backend**: .NET 10.0 Web API
- **Route Optimization**: Google Maps Route Optimization API v1.4.0
- **Database**: SQLite with Entity Framework Core
- **API Documentation**: OpenAPI/Swagger integration
- **Development Environment**: ASP.NET Core with HTTPS support
  Prerequisites
- .NET 10.0 SDK
- Google Maps Route Optimization API key
- Git for version control
